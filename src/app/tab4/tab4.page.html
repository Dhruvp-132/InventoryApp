Privacy & Security Privacy & Security Data Privacy

This app connects to a remote RESTful API. It stores only inventory data (item name, category, stock details, supplier information). No authentication credentials or sensitive personal information are collected.

Data Transmission

All communication uses HTTP(S) requests with JSON data. The backend server enforces constraints such as unique item names. Inventory records are stored in a remote database.

User Responsibilities

Do not enter real confidential information. Use fictional example data for assessment work. Always close the app when finished, especially on shared computers.

Limitations

This is an academic exercise, not for production use. It does not include advanced security controls such as role-based access control or encryption at rest.
